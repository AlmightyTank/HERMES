# HERMES 1.2.0 Release Notes

1.2.0 adds confirmed completed craft collection in the Crafts workspace.

## New In 1.2.0

- Collect one completed regular Hideout craft after confirmation.
- Collect all currently completed regular crafts after confirmation.
- Preview station, crafted item, output quantity, stash fit, and expected result before confirming.
- Reject stale confirmations when a craft is missing, already collected, no longer complete, unsupported, or cannot fit in stash.

Automatic craft collection is not included. The player still confirms every collection.
